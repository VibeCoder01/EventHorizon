# **App Name**: Event Horizon

## Core Features:

- Event Log Selection: Allow the user to select log files from the local machine (Windows or Linux).
- Event Log Upload: Allow users to upload Windows or Linux logs.
- Data Parsing: Parse the selected and uploaded event log files to extract event data, including Windows and Linux formats.
- Event Timeline Visualization: Generate a visual timeline of events with filtering options by event type (Information, Warning, Error, Critical, Verbose, Application, System).
- Event Table Display: Display the raw event logs in a tabular format, allowing users to view detailed information for each event.
- Significant Findings: Display any significant findings derived from the event logs.

## Style Guidelines:

- Primary color: Dark midnight blue (#192A56) to evoke a sense of security and technical sophistication.
- Background color: Near-black (#0A0E1A) for high contrast and to emphasize the timeline visualization.
- Accent color: Electric blue (#7DF9FF) for interactive elements and important data points, ensuring visibility against the dark background.
- Body and headline font: 'Inter', a sans-serif font for its modern, neutral appearance, which will offer clear legibility and readability.
- Use clear, minimalist icons to represent different event types and severity levels.
- Divide the screen into three sections: timeline visualization at the top, detailed event list in the middle, and summary/insights at the bottom.
- Use subtle animations to highlight important events on the timeline and to provide feedback on user interactions.
